# DongJinDam (동진담) - Landing Page

동양의 진귀함과 현대 과학이 만나는 머스크 기반 웰니스 브랜드 동진담의 공식 랜딩페이지입니다.

## 🌟 프로젝트 소개

DongJinDam은 머스크를 핵심으로 삼아 과학적 검증과 프리미엄 브랜딩을 결합한 웰니스 브랜드입니다. 이 랜딩페이지는 동양의 전통적 미학과 현대적 디자인을 조화롭게 표현하여 브랜드의 정체성을 효과적으로 전달합니다.

## 🎨 주요 섹션

- **Main Hero Section**: 풀스크린 히어로 섹션과 그라데이션 텍스트 효과
- **Introduction**: 브랜드 소개 및 핵심 가치 전달
- **Image Showcase**: 2x2 분할 그리드 레이아웃으로 시각적 임팩트 제공
- **Philosophy**: 브랜드 철학을 다이어그램과 함께 표현
- **Slogan**: 브랜드 슬로건 및 비전 제시
- **Brand Positioning**: 4개의 브랜드 포지셔닝 사례 소개
- **Interior Space**: 프리미엄 공간 이미지 갤러리
- **Footer**: 네비게이션 링크 및 연락처 정보

## 🛠️ 기술 스택

- **Backend**: Node.js + Express
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Google Fonts (Montserrat, Poppins, Noto Sans KR)
- **Design**: 다크 테마, 그라데이션, 글라스모피즘

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```

서버가 실행되면 브라우저에서 `http://localhost:3000`으로 접속하세요.

## 📁 프로젝트 구조

```
parfait-page/
├── server.js              # Express 서버
├── package.json           # 프로젝트 설정
├── public/
│   ├── index.html        # 메인 HTML
│   ├── css/
│   │   ├── style.css     # 메인 스타일시트
│   │   └── footer.css    # 푸터 스타일시트
│   ├── js/
│   │   └── script.js     # 클라이언트 스크립트
│   └── assets/           # 이미지 및 미디어 파일
```

## ✨ 주요 기능

### 인터랙션 & 애니메이션
- 스크롤 기반 섹션 애니메이션 (Intersection Observer)
- 부드러운 스크롤 네비게이션
- 호버 효과 및 이미지 줌
- 패럴랙스 효과

### 반응형 디자인
- 모바일, 태블릿, 데스크톱 완벽 대응
- 유연한 그리드 레이아웃
- 디바이스별 최적화된 폰트 크기

### 성능 최적화
- 이미지 최적화
- CSS 애니메이션 하드웨어 가속
- 효율적인 스크롤 이벤트 처리

## 🎯 디자인 컨셉

- **색상 팔레트**: 오렌지 톤 (`hsl(16.8deg 90.09% 56.47%)`), 다크 테마, 베이지 그라데이션
- **타이포그래피**: Montserrat (영문), Noto Sans KR (한글)
- **스타일**: 프리미엄 다크 모드, 글라스모피즘, 그라데이션 효과
